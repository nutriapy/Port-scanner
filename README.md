# Port-scanner
Script simple en Python que escanea puertos y muestra banners.

Este es un script que hice para escanear puertos de una IP.
Muestra qué puertos están abiertos, qué servicio puede tener y también intenta sacar el banner si responde.

Lo hice para practicar Python y temas de redes.

Cómo usarlo

Solo corre:

'''python custom_port_scaner.py'''

Pones la IP y listo.
